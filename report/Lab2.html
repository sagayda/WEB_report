Тема: КАСКАДНІ ТАБЛИЦІ СТИЛІВ. СЕЛЕКТОРИ .ІДЕНТИФІКАТОРИ. СТИЛЬОВЕ ОФОРМЛЕННЯ ТЕКСТОВИХ ЕЛЕМЕНТІВ В HTML-ДОКУМЕНТАХ.

Мета: придбати практичні навички роботи з селекторами, ідентифікаторами, списками, різноманітними властивостями кольору і фону, зовнішними та внутрішними відступами, плаваючими елементами, оформленням текстових елементів.

Існує три способи підключення стилів CSS до HTML:

  • Вбудовані стилі. CSS-правила додаються безпосередньо в HTML-код, в атрибуті style тегу <style>. Цей спосіб є найпростішим, але його не рекомендується використовувати для великих проектів, оскільки він може ускладнити читання і редагування коду. 
    <html lang="en">
    <head>
        <title>My Website
        <style>
            h1 {
                color: red;
                font-size: 2em;
            }
        </style>
    </head>
</html>
  • Зовнішні стилі. CSS-правила зберігаються в окремому файлі з розширенням .css. Файл CSS додається до HTML-документу за допомогою тегу <link>. Цей спосіб є більш зручним і кращим для організації коду. 
    <html lang="en">
    <head>
        <title>My Website</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <h1>This is a heading</h1>
    </body>
</html>
  • Стилі, імпортовані за допомогою @import. CSS-правила можна імпортувати з інших CSS-файлів за допомогою директиви @import. Цей спосіб є аналогічним використанню зовнішніх стилів, але дозволяє імпортувати стилі з інших файлів, не створюючи для них окремі теги <link>. 
    <html lang="en">
    <head>
        <title>My Website
        <style>
        @import url("styles.css");
        </style>
    </head>
    <body>
        <h1>This is a heading</h1>
    </body>
</html>

Селектор тегу - це тип селектора CSS, який використовується для вибору елементів на основі їх імені тегу. Селектори тегу - це найпростіший тип селектора CSS. Вони часто використовуються для стилізації елементів за їхньою функцією на сторінці. Наприклад, можна використовувати селектор тегу h1 для стилізації заголовків, а селектор тегу p для стилізації абзаців. 

body {
    background-color: #664C3B;
}

Селектор класу - це тип селектора CSS, який використовується для вибору елементів на основі їх класу. Клас - це атрибут тегу, який можна використовувати для групування елементів. Селектор класу складається з точки (.) і імені класу.
Селектори класу - це потужний інструмент, який дозволяє стилізувати елементи за їхнім змістом або призначенням. Наприклад, можна використовувати селектори класу для стилізації всіх елементів, які є посиланнями, або для стилізації всіх елементів, які є зображеннями. 

.attic-link a {
    display: inline-block;
    font-size: medium;
    color: #d6b6a1;
    text-decoration: none;
    border-bottom: dotted #d6b6a1 3px;
    padding: 0;
    cursor: pointer;
    transition: color 0.5s ease-out, border-bottom 0.5s ease-out;
}

.attic-link a:hover {
    color: #EAD9CE;
    border-bottom: solid #EAD9CE 3px;
    transition: border-bottom 0.5s ease-out;
}

.attic-link i {
    margin-right: 4px;
}

Селектор ідентифікатора - це тип селектора CSS, який використовується для вибору елементів на основі їхнього ідентифікатора. Ідентифікатор - це атрибут тегу, який може використовуватися для унікальної ідентифікації елемента. Селектор ідентифікатора складається з знака решітки (#) і імені ідентифікатора.
Селектори ідентифікатора - це найточніший тип селектора CSS. Вони дозволяють стилізувати тільки один елемент на сторінці. 

#margin1 {
    margin: 0px 12px;
}

<div class="attic-button-group horizontal-list-group">
    <ul>
        <li class="attic-animated-link" id="margin1">
            <a href="#">UKR
        </li>
    </ul>
</div>

Крім селекторів тегу, класу і ідентифікатора, існує ще кілька типів селекторів CSS:

  • Селектори псевдокласів - це тип селектора CSS, який використовується для вибору елементів на основі їхнього стану або поведінки. Наприклад, селектор псевдокласу :hover буде вибирати елементи, які знаходяться під курсором миші.

  • Селектори псевдоелементів - це тип селектора CSS, який використовується для вибору елементів, які не існують фізично на сторінці. Наприклад, селектор псевдоелементу ::after буде вставляти новий елемент після вибраного елемента.

  • Селектори атрибутів - це тип селектора CSS, який використовується для вибору елементів на основі їхніх атрибутів. Наприклад, селектор атрибута [href] буде вибирати всі елементи, які мають атрибут href.

  • Селектори комбінатори - це тип селектора CSS, який використовується для вибору елементів на основі їхніх взаємозв'язків. Наприклад, селектор комбінатора > буде вибирати всі елементи, які є прямими потомками вибраного елемента.

Комбінуючи ідентифікатори, стилі та властивості можна створювати незвичайні та красиві елементи на сторінці, додавати анімації, переходи та інше.
