Тема: Веб-додаток меню ресторану з можливістю оцінки страв.

Мета: Написання стайту для ресторану, на якому можна буде переглядати страви та залишати про них відгуки. Додавання до сайту необхідних елементів для зручного користування. Додавання зручної системи данних для додавання та вилучення страв з переліку.

Об'єкт дослідження: Система відображення інформації про страви та система відгуків.

Предмет дослідження: Аналіз роботи існуючих онлайн-сервісів ресторанів.

Постановка задачі: Розробка веб-додатку, за допомогою якого користувач переглядає доступні для замовлення страви, ціни на них, та відгуки інших відвідувачів. Розробка системи відображення та вибору страв за критеріями користувача. Розробка системи данних для їх оновлення та актуалізації.

Бізнес логіка:

Користувач:

  • Реєстрація
  • Авторизація
  • Перегляд страв
  • Пошук страв
  • Вибір режиму відображення
  • Перегляд відгуків відвідувачів
Користувач (зареєстрований):
  • Оцінка замовленої страви
  • Написання відгуку
Адміністратор:
  • Додавання страв
  • Видалення страв
  • Редагування інформації про страви
  • Модерація відгуків

<div class="top">
Цей тег використовується для створення контейнера (блоку) з ідентифікатором класу "top".

<table>
Тег використовується для створення таблиці на веб-сторінці.

<tbody>
Тег вказує на початок тіла таблиці. Це зазвичай необов'язковий тег, але він часто використовується для групування рядків таблиці.

<tr>
Тег використовується для створення рядків в таблиці.

<th>
Тег використовується для створення заголовкових (шапки) комірок в таблиці. В цьому випадку, заголовки комірок містять посилання.

Тег для вставки зображень в веб-сторінку.

src
Атрибут, який вказує шлях до зображення, яке відображатиметься на сторінці.

width
Атрибут, який вказує ширину зображення відносно його оригінального розміру.

height
Атрибут, який вказує висоту зображення відносно його оригінального розміру.

<ul>
Цей тег позначає початок ненумерованого списку (список з маркерами, наприклад, крапками чи знаками списку)

<li>
Цей тег позначає елемент списку (список пунктів). В даному випадку, він використовується для кожного елементу меню.

<b>
Цей тег використовується для виділення тексту як жирний (bold).

Додатково, у вас також вкладені в <li> такі теги:
<ol>
Цей тег позначає початок нумерованого списку (список з номерами).

<li>
Цей тег використовується вкладено в <ol> для нумерації елементів у списку.
